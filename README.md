# Inventory-App-Android
CS-360 Inventory App Android

Inventory Management Application
Overview

The Inventory Management Application is a fully functional Android mobile app built using Java and Android Studio. The app allows users to securely create an account, log in, and manage inventory items using a persistent SQLite database. It also includes optional SMS notifications that alert users when inventory levels fall below a defined threshold.

This project was developed as part of CS 360 Mobile Architect and Programming at Southern New Hampshire University and demonstrates a complete mobile development lifecycle from design to launch planning.

Features

Secure user login and account creation

Persistent local database using SQLite

Full CRUD functionality:

Create inventory items

Read and display items in a RecyclerView grid

Update item details

Delete items

Runtime SMS permission handling

Conditional SMS alerts for low inventory

Clean, user-centered interface design

Defensive coding to ensure functionality even when permissions are denied
